What is the difference between `var`, `let`, and `const`?
In JavaScript, var, let, and const are used for variable declaration, but they differ in scope and mutability: var is function-scoped and can be re-declared and reassigned, while let and const are block-scoped, let can be reassigned but not re-declared, and const cannot be reassigned or re-declared.

How JavaScript Promises work
A Promise in JavaScript is an object that represents the eventual completion (or failure) of an asynchronous operation. It helps handle async code without using callbacks, making the code more readable.
How closures work in JavaScript
A closure in JavaScript is created when a function remembers and has access to variables from its outer (lexical) scope, even after the outer function has finished executing.
